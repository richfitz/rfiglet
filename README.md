## Fonts

There are many FIGlet compatible fonts; to keep things small, this package only includes the included by FIGlet

* banner
* big
* block
* bubble
* digital
* ivrit
* lean
* mini
* mnemonic
* script
* shadow
* slant
* small
* smscript
* smshadow
* smslant
* standard
* term
